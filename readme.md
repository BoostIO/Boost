## Roadmap

- Browser app
- Desktop app(Windows, mac and Linux)
- Mobile app(Android and iOS)
- Cloud storage
- File attachments(Image, Video and any kind of files)
- MDX (TBA)

## Build

### Web app

```sh
# Install deps
npm i

# Compile script
npm run build

# Deploy to now.sh
now
```

### Electron app

```sh
# Install deps
npm i

# Compile script
npm run build

# Package app
npm run pack
```
