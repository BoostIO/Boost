## Build

### Web app

```sh
# Install deps
npm i

# Compile script
npm run build

# Deploy to now.sh
now
```

### Electron app

```sh
# Install deps
npm i

# Compile script
npm run build

# Package app
npm run pack
```

### Deploy release

Prepare .env

```sh
# Usage analytics
AMPLIFY_AUTH_IDENTITY_POOL_ID=
AMPLIFY_AUTH_REGION=
AMPLIFY_PINPOINT_APPID=
AMPLIFY_PINPOINT_REGION=

# Certificates info for Windows app
CSC_LINK=
CSC_KEY_PASSWORD=

# Deploy token
GH_TOKEN=
```

Run the below scripts on each OS

```sh
# Bump version
npm version *

# Install deps
npm i

# Compile script
npm run build

# Pack and Release
npm run release
```

Now you will have a release draft with the installer attachemnts. Update its content and click publish. Then the update becomes availble.
